# e-plantShopping - Paradise Nursery

e-plantShopping (Paradise Nursery) is a React + Redux based online plant shopping application that allows users to browse indoor plants, add them to cart, and manage quantities dynamically.

## Features

- Browse plants grouped by categories
- View plant name, price, and image thumbnails
- Add plants to shopping cart
- Increase or decrease item quantities in cart
- Remove items from cart
- View total cart amount dynamically
- Continue shopping and checkout flow

## Technologies Used

- React (Vite)
- Redux Toolkit
- React Router DOM
- JavaScript (ES6+)
- CSS

## Project Structure

- ProductList.jsx → Displays plant categories and products
- CartItem.jsx → Handles shopping cart functionality
- CartSlice.jsx → Redux state management for cart
- App.jsx → Main navigation and routing logic
