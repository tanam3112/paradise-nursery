# Paradise Nursery

Paradise Nursery is an online plant shopping application built with React and Redux.

## Features

- Browse plants by category
- Add plants to cart
- Update item quantities
- Remove items from cart
- View total cart value
- Responsive user interface

## Technologies Used

- React
- Redux Toolkit
- React Router DOM
- Vite

## Installation

```bash
npm install
npm run dev
```
